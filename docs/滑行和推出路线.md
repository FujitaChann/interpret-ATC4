点属性中
```ini
0x00000000 = 普通的点 
0x00000002 = 停机坪 
0x00840000 = 自滑点 
0x00040000 = 出现在机坪中间的点（不清楚干啥的） 
0x00200000 = 滑行道与机坪的连接点
```
